# Webpack Template

## Features

- Basic files support (HTML, CSS, Stylus, JS, IMages, font)
- Extracted CSS
- Transpilled JS (with Babel)
- Sourcemaps
- Clean dist
- Dev server (with local network support)
- Static Files
- Caches breaker

## Instructions

Install dependencies

```npm i```

Launch dev server

```npm run dev```

Build for production

```npm run build```
